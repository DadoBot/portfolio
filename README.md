# Davide's Professional Portfolio

Welcome to my professional portfolio repository! This is a minimalist, responsive, single-page web application built with **React** and **Vite**. It is designed with a premium, high-fidelity dark-mode aesthetic inspired by the editorial, masonry-style layouts of modern web design.

**Live Demo:** [https://DadoBot.github.io/portfolio](https://DadoBot.github.io/portfolio)

## Features

- **Minimalist & Premium Design:** Features a bespoke, dark-themed UI with tailored typography, smooth gradients, and carefully crafted components.
- **Bilingual Support:** Full English/Italian translation capability using a seamless toggle.
- **Dynamic Projects Showcase:** Specifically highlights deep-dives into "Marvelli" and "Home-server", outlining roles, technological approaches, and key learnings.
- **Responsive Layout:** Fluidly adapts to mobile, tablet, and desktop viewports, using a custom grid masonry system for content.
- **Optimized Performance:** Fast loads and a smooth experience powered by Vite.

## Built With

- [React](https://reactjs.org/) (v19)
- [Vite](https://vitejs.dev/)
- Vanilla CSS (Custom Design System)
- [gh-pages](https://github.com/tschaub/gh-pages) (Deployment)

## Getting Started

To run this project locally, simply follow these steps.

### Prerequisites

- Node.js installed

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/DadoBot/portfolio.git
   cd portfolio
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`.

## Deployment

This portfolio is automatically deployed to GitHub Pages. To deploy manually:

```bash
npm run deploy
```

This will run the build script and push the optimized `/dist` folder to the `gh-pages` branch.
